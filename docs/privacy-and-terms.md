## 수집 데이터

### users 테이블 (회원가입 시)

| 항목           | 설명                         |
| -------------- | ---------------------------- |
| 이메일         | 로그인 및 식별자             |
| 닉네임         | 소셜 프로필명 또는 직접 입력 |
| 가입 방법      | google / kakao / email       |
| 약관 동의 일시 | 동의 완료 시 기록            |

### diary 테이블 (일기 작성 시)

| 항목      | 설명                        |
| --------- | --------------------------- |
| 작성 날짜 | YYYY-MM-DD                  |
| 일기 내용 | 텍스트                      |
| 감정 극성 | POSITIVE / NEGATIVE / UNSET |
| 감정 강도 | 1~5단계 (선택)              |
| 감정 태그 | 서비스 제공 태그 중 선택    |

### star / constellation 테이블 (자동 생성)

| 항목          | 설명                   |
| ------------- | ---------------------- |
| 별(Star) 기록 | 일기 작성 시 자동 생성 |
| 별자리 통계   | 기간별 일기 작성 횟수  |

### 수집하지 않는 정보

- 위치 정보, 연락처, 사진/미디어, 기기 식별자, 결제 정보
